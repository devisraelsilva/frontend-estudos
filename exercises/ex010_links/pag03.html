Links Externos

Terceira página

Tabalharemos com a tag A (Anchor/Ancora), para anexarmos links de sites que queremos redirecionar, exemplo abaixo:

"Em meus estudos de desenvolvimento WEB, estou assistindo as aulas e acompanhando os materiais PDF, disponibilizados pelo Prof. Gustavo Guanabara, da plataforma Curso em Video."

Abaixo analíse como está escrita a frase acima:

        
    "<p>Em meus estudos de desenvolvimento WEB, estou
        <a
        href="https://www.youtube.com/watch?v=Ejkb_YpuHWs&list=PLHz_AreHm4dkZ9-atkcmcBaMZdmLHft8n"
        target="_blank"
        rel="external"
        >assistindo as aulas</a
        >
        e
        <a
        href="https://github.com/gustavoguanabara/html-css/tree/master/aulas-pdf"
        target="_blank"
        rel="external"
        >acompanhando os materiais PDF</a
        >, disponibilizados pelo <abbr title="Professor">Prof.</abbr> Gustavo
        Guanabara, da plataforma
        <a href="https://www.cursoemvideo.com/" target="_blank" rel="external"
        >Curso em Video</a
        >.</p>"

Podemos Analisar os atributos da tag A:

Target="" - setamos se queremos abrir o link na aba atual ou uma nova aba.
Rel="" - para referir se é um link para página externa(external), ou nofollow.
