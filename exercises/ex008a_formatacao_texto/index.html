Principais Formatações

Neste documento, realizaremos o estudo em formatação dos textos, utilizando tags semânticas (com significado para a linguagem), e não semânticas (sem significado, apenas forma).

Visualmente são iguais.

Negrito / Destaque

Nesta frase estamos utilizando o TERMO EM NEGRITO, utilizando a tag B (não semântica).

Já nesta frase, estamos utilizando o TERMO COMO DESTAQUE, utilizando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase estamos usando o TERMO EM ITÁLICO, utilizando a tag I (não semântica).

Já nesta frase, utilizamos o TERMO EM ÊNFASE, utilizando a tag EM (semântica).

Texto marcado / Marca texto

Podemos criar também UM TEXTO MARCADO, utilizando a tag MARK.

E nesta frase, realizamos a alteração da cor de marcação, dentro da tag MARK, podemos analisar O TEXTO MARCADO no final.

Apenas para confirmação AQUI ESTÁ MARCADO com a cor padrão, configurada na HEAD.

Texto grande e texto pequeno

Nesta frase conseguimos analisar a diferença de TEXTO GRANDE para TEXTO PEQUENO.

Texto deletado

Podemos aplicar também uma marcação, COMO TEXTO EXCLUÍDO.

Texto inserido

Podemos aplicar um TEXTO COMO INSERIDO, semelhante a sublinhado (semântica).

Conseguimos manter UM TEXTO COM SUBLINHADO, utilizando a tag U (não semântica).

Texto sobrescrito

Podemos também formatar coisas, com texto menor superior, como por exemplo: x20+4; x7; b.45a+c.

Texto subscrito

Podemos também formatar coisas, com texto menor inferior, como por exemplo: H2O; H2N; CH3; CO2.
